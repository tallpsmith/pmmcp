## Summary

Add a distributed specialist agent model for concurrent performance investigation. Two new MCP prompts ‚Äî `specialist_investigate` (parameterized by subsystem, carrying deep domain expertise) and `coordinate_investigation` (parallel dispatch + synthesis) ‚Äî replace the shallow namespace-hint approach in `investigate_subsystem`. Secondary fix: bump `pcp_search` default limit from 20‚Üí50. Update `session_init` to reference the coordinator as the investigation entry point.

No new tools needed. No new dependencies. Pure prompt-layer work plus a one-line default value change.

## Design Decisions (from research.md)

| ID | Decision | Rationale |
|----|----------|-----------|
| R1 | Prompts, not tools | Orchestration happens in LLM reasoning; no new tools needed |

| R3 | 5-8 concrete heuristics per subsystem | SC-002 requires depth beyond namespace hints |

| R7 | Search limit 20‚Üí50 | Spec's explicit value; secondary improvement only |
